3D Super Tic Tac Toe

About

Inspired by 3D Tic Tac Toe created by Nicolas Chan, 3D Super Tic Tac Toe takes the game to a whole new level. With 729 spaces to conquer, you can no longer block your opponent with one simple move. The minimum amount of moves required to win the game is 15, contrary to 5 moves in normal Tic Tac Toe. Furthermore, if we do not consider the final move, this game has 1552472747942425344000 possible combinations! (Total possible combinations for the final move requires the computing power of a quantum or super computer to calculate)

Rules

Get 9 blocks in a row to win. Note that you can also go diagonally across the entire playing field. There are 729 paths to win this game

The red transparent block indicates the section where you are allowed to place your block. This block moves each round based on player move to add complexity to the game
